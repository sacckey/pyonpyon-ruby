# Pyon Pyon Ruby

レトロゲームエンジン Reight (R8) で作ったアクションゲームです。

## スクリーンショット
![Image](https://github.com/user-attachments/assets/e1aa3631-9768-4c6f-9a4b-cd42d783cf01)

## インストール

```sh
$ git clone https://github.com/sacckey/pyonpyon-ruby.git
$ cd pyonpyon-ruby
$ bundle install
$ bundle exec r8 pyon
```

## 遊び方

```sh
$ bundle exec r8 pyon
```

方向キー(← →): 左右移動  
スペースキー: ジャンプ
